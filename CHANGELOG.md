# 变更日志  

## [1.6.0] - 2026-03-10
### 新增功能
- 新增字段分组导航，支持按分组折叠查看，便于在大字段表中快速定位目标字段。
- 将原有分组导航升级为更完整的目录面板，补充更清晰的层级浏览与入口组织。
- 新增图片状态接口请求拦截能力，可在插件弹窗中开关 `GET /api/getAllImagesByStatus1` 的拦截。

### 交互与体验改进
- 重构导出按钮交互，拆分表格导出与 Markdown 导出图标组件，简化状态管理并改进按钮反馈。
- 优化弹窗配置与内容脚本联动，增强目录面板与请求拦截开关的可用性。

### 品牌与资源更新
- 更新插件品牌形象与 Logo 资源，替换主 Logo 与多尺寸图标。
- 新增品牌理念文档，以及展示图、图形标识、iOS 图标等一组设计资产。

### 工程与文档
- 将打包脚本迁移至 `scripts/` 目录，并新增 `package` 命令，统一构建入口。
- 补充 README 中的品牌展示内容，并完善版本变更记录。
