# 聚美美 Chrome 插件安装指南

## 插件简介

聚美美是一个专为聚源数据字典（https://dd.gildata.com/）设计的Chrome浏览器扩展，提供数据字典的可视化增强功能。

### 主要功能
- 字段类型显示区分（类似DBeaver风格）
- 业务唯一性字段加粗显示
- 鼠标悬停高亮效果

## 系统要求

- Google Chrome 浏览器 88+ 版本
- 支持Manifest V3的Chrome版本
- 需要访问 https://dd.gildata.com/ 网站

## 故障排除

### 插件未生效
1. 确认插件已启用：在chrome://extensions/页面检查插件状态
2. 刷新目标网页：按F5或Ctrl+R刷新聚源数据字典页面
3. 检查网站URL：确保访问的是 https://dd.gildata.com/ 域名

**[占位符：插件状态检查截图]**
*说明：显示如何在扩展管理页面检查插件启用状态*

### 样式显示异常
1. 清除浏览器缓存：Ctrl+Shift+Delete
2. 重新加载插件：在扩展管理页面点击刷新按钮
3. 检查控制台错误：按F12打开开发者工具查看错误信息

**[占位符：控制台检查截图]**
*说明：显示如何使用浏览器开发者工具检查错误*

1. 插件已获得"活动标签页"权限
2. 网站URL匹配插件配置的域名
